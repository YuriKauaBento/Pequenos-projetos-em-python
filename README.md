Este repositório contém alguns dos meus estudos em python.
Cada pasta contêm um pequeno projeto separado aplicando os conhecimentos que adquiri.

d001 - Um exercício simples e introdutório a POO. A classe cadastra um funcionário e sua função apresentar() o apresenta.

d002 - Este exercício, utilizando a biblioteca rich, constrói etiquetas simples de produtos com sua descrição e valor.

d003 - Este programa simples faz um cálculo estimativo dos gastos e da divisão de custos de um churrasco.

d004 - Simula as passagens de página de um livro.

d005 - Registra os jogos favoritos de vários jogadores.

d006 - Simula o funcionamento de uma caneta.
